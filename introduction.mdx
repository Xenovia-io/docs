# Govern agent runtime

Xenovia is an operator-first control plane for agentic systems. It helps teams see what agents are doing, control what tools they can access, enforce policy decisions with approvals, and preserve evidence for review and remediation.

<Callout type="info">
  Built for production operators: legible decisions, policy-backed control, and fast remediation.
</Callout>

## Core lifecycle

<Steps>
  <Step title="Intent">
    An agent requests a tool action or workflow change.
  </Step>
  <Step title="Comparison">
    Xenovia compares the request against scope, ownership, and policy.
  </Step>
  <Step title="Decision">
    Runtime enforcement allows, blocks, or escalates the action.
  </Step>
  <Step title="Trace">
    Xenovia records decision evidence and execution details.
  </Step>
  <Step title="Operator action">
    Teams review, approve, reroute, or remediate with full context.
  </Step>
</Steps>

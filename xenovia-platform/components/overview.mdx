# Xenovia Platform Components

Xenovia is organized into several interconnected components, each designed to address specific aspects of the AI development lifecycle. This overview will help you understand how these components work together to create a comprehensive MLOps solution.

<Callout type="info">
  Each component can be used independently, but they are designed to work together seamlessly for maximum efficiency.
</Callout>

## Component Relationships

Understanding how these components interact is key to leveraging the full power of Xenovia:

<Steps>
  <Step title="Build Phase">
    Start with Agent Creation, Prompts, and Knowledge Management to build your AI solution
  </Step>
  <Step title="Test Phase">
    Use Testing, Agent Execution, and Analytics to validate performance
  </Step>
  <Step title="Optimize Phase">
    Leverage Fine-tuning, Tools, and Integrations to enhance capabilities
  </Step>
  <Step title="Deploy Phase">
    Utilize Deployment, Security, and Collaboration to publish and manage
  </Step>
  <Step title="Monitor Phase">
    Rely on Dashboard, Analytics, and Memory Management to track operations
  </Step>
</Steps>

Explore each component in detail through the navigation menu to understand its capabilities and how to use it effectively.
